## 简介

Everything Claude Code 是一个Claude Code 的生产力增强工具包，包含：
- **66个技能 (Skills)** - 领域专业知识和工作流
- **40个命令 (Commands)** - 快捷操作和自动化
- **16个代理 (Agents)** - 专家级AI助手
- **多套规则 (Rules)** - 代码规范和最佳实践

---

## 快速开始

### 安装/更新

```powershell
# 方法1: 自动更新（推荐）
运行 auto-update.ps1

# 方法2: 手动更新
cd "D:\AI Project\everything-claude-code"
git fetch upstream main
git reset --hard upstream/main
```

### 使用方式

在 Claude Code 中输入：
- `/命令名` - 执行命令
- `@代理名` - 委托给代理
- 技能会自动根据上下文触发

## 五、自动更新

### 配置

工具包已配置自动更新：

1. **SessionStart Hook** - 每次启动检查更新
2. **Windows定时任务** - 每周一9点自动更新

### 手动更新

```powershell
# 运行更新脚本
D:\AI Project\everything-claude-code\auto-update.ps1

# 查看更新日志
type D:\AI Project\everything-claude-code\update.log
```

### 设置定时任务

```powershell
# 以管理员运行
D:\AI Project\everything-claude-code\setup-scheduled-update.bat
```
