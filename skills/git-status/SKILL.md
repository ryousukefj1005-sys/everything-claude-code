# Git 状态查看技能

## 描述

显示当前 Git 仓库的详细状态，包括分支、改动文件、未提交内容等。

## 使用场景

- 查看当前工作进度
- 了解哪些文件被修改
- 检查是否有未提交的改动

## 执行步骤

1. 检查当前是否在 Git 仓库中
2. 显示当前分支
3. 显示未提交的改动
4. 显示未跟踪的文件
5. 显示最近 3 条提交记录

## 参数

无

## 示例

```
/git-status
```

## 输出格式

```
当前分支: master
未提交的改动:
  M  core/analysis.py
  M  config/settings.py
未跟踪的文件:
  ?? new_file.py
最近提交:
  abc1234 feat: 添加新功能
  def5678 fix: 修复 Bug
  ...
```
