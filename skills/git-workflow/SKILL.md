# Git 快速同步技能

## 描述

快速提交当前项目的所有改动并推送到远程仓库。适用于开发完成一个功能后的快速备份。

## 使用场景

- 写完一个功能后快速备份
- 切换电脑前同步代码
- 每日工作结束前备份

## 执行步骤

1. 检查当前是否在 Git 仓库中
2. 显示当前 Git 状态
3. 添加所有改动文件
4. 询问用户提交信息（提供默认选项）
5. 执行提交
6. 推送到远程仓库
7. 显示成功信息

## 参数

- `message` (可选): 提交信息，如果不提供则询问用户

## 示例

```
/git-sync
/git-sync "feat: 添加新功能"
```

## 注意事项

- 只在 Git 仓库中工作
- 如果没有远程仓库，只执行本地提交
- 推送前会先拉取远程更新（避免冲突）
