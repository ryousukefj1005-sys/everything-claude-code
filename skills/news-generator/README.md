# News Generator Skill

24小时全球新闻深度分析报告生成器

## 功能

自动生成包含以下内容的深度新闻分析报告：
- 📋 执行摘要（核心关键词、最重要事件、行动建议）
- 🌍 24小时新闻深度扫描（20-25条情报模块）
- 📊 AI宏观总结（核心数据、趋势分析、风险评估）
- 🛡️ 个人行动指南（分人群建议、机遇与风险）

## 使用方法

在 Claude Code 对话中输入以下任一关键词：
- "刷新新闻"
- "24小时新闻"
- "更新新闻"
- "生成新闻报告"

## 输出

报告将保存到 报告内容 文件夹：
```
d:\AI Project\24小时新闻提示词\报告内容\[生成时间].md
```

文件名格式：`xxxx年xx月xx日xx时xx分xx秒.md`
