# News Generator Skill

生成24小时全球新闻深度分析报告

## 触发关键词

- "刷新新闻"
- "24小时新闻"
- "更新新闻"
- "生成新闻报告"
- "新闻分析"
- "搜索24小时新闻"

## 描述

当用户输入触发关键词时，自动使用 `AI新闻深度分析提示词-每日版v3.md` 中的完整提示词内容，生成一份新的24小时全球新闻深度分析报告。

## 执行步骤

1. 首先使用 Bash 工具查询真实当前时间：
   ```bash
   date +"%Y-%m-%d %H:%M:%S %Z"
   ```

2. 读取提示词模板文件：
   ```
   d:\AI Project\24小时新闻提示词\AI新闻深度分析提示词-每日版v3.md
   ```

3. 按照提示词要求执行完整流程

4. 生成报告并保存到 scripts 文件夹：
   - 路径：`d:\AI Project\24小时新闻提示词\scripts\`
   - 文件名格式：`xxxx年xx月xx日xx时xx分xx秒.md`
   - 例如：`2026年01月27日16时45分30秒.md`

## 输出文件路径

```
d:\AI Project\24小时新闻提示词\报告内容\[生成时间].md
```

文件名格式：`xxxx年xx月xx日xx时xx分xx秒.md`

## 注意事项

- 必须先查询真实时间，不能假设或猜测
- 严格使用提示词模板中的完整格式
- 确保新闻时效性在24小时内
- 报告生成后告知用户文件保存位置
