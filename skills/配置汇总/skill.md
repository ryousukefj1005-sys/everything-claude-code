# 配置汇总 Skill

生成 Claude Code 配置汇总报告

## 触发关键词

- "配置汇总"
- "生成配置汇总"
- "配置报告"
- "配置检查"
- "查看配置"

## 描述

当用户输入触发关键词时，自动执行配置汇总脚本，生成包含以下内容的完整报告：
- Python 环境信息
- MCP 服务器配置
- Skills 配置
- 全局配置文件
- 依赖包列表
- 文件状态检查

## 执行步骤

1. 运行配置汇总脚本：
   ```cmd
   "D:\AI Project\配置管理-脚本\配置脚本生成.bat"
   ```
   或双击 `配置脚本生成.bat` 文件运行

2. 脚本会自动生成报告到：
   ```
   D:\AI Project\配置管理-报告\配置汇总.md
   ```

3. 告知用户报告已生成，显示文件路径

## 相关文件

- 脚本位置：`D:\AI Project\配置管理-脚本\make_config_summary.ps1`
- 报告输出：`D:\AI Project\配置管理-报告\配置汇总.md`
