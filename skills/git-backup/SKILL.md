# Git 多项目备份技能

## 描述

批量备份 AI Project 下的所有子项目到各自的 GitHub 仓库。

## 使用场景

- 定期批量备份多个项目
- 确保所有项目都已推送
- 切换电脑前统一备份

## 执行步骤

1. 扫描 AI Project 目录
2. 检查每个子项目的 Git 状态
3. 询问要备份哪些项目
4. 依次执行 git add、commit、push
5. 显示备份结果汇总

## 参数

- `projects` (可选): 指定要备份的项目列表，不指定则扫描所有

## 示例

```
/git-backup
/git-backup 港股通数据,人生OS V2.1
```

## 输出格式

```
扫描到 5 个 Git 仓库:
  ✓ 港股通数据 - 有未提交改动
  ✓ 人生OS V2.1 - 已是最新
  ✓ AI量化 - 有未提交改动
  ✓ 炒股Game - 已是最新
  ✗ 家庭账本 - 未初始化 Git

选择要备份的项目: [1] 港股通数据 [2] AI量化 [3] 全部
...
备份完成:
  ✓ 港股通数据 - 已推送 3 个提交
  ✓ AI量化 - 已推送 1 个提交
```
